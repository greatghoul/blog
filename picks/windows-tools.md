老狗用过的觉得不错的 Windows 软件

### SpaceSniffer - Windows 下的免费硬盘清理工具
https://www.uderzo.it/main_products/space_sniffer/features.html

扫描硬盘后，按照文件夹大小，可视化为大小不一的区块，非常直观，如果你想要下载新游戏，又苦于硬盘空间不足，可以考虑下这个工具，非常好用。

<img width="2048" height="799" alt="Image" src="https://github.com/user-attachments/assets/5b72304a-7ebe-4961-b424-01f3077396d7" />

### ContextMenuManager - 开源 Windows 右键菜单管理工具
https://www.anl.gg/post/292-context-menu-manager/

开源软件，官方说明是支持到 Win10, 但其实 Win11 亲测可用。可以添加/编辑/删除/启用/禁用各项右键菜单，还支持新增/打开方式/发送到这些折叠菜单的项目。
