## 画板

### Sketchpad - 简洁又强大的在线画板工具

https://sketch.io/sketchpad/

> 虽然比较小巧简洁，但麻雀虽小，五脏俱全，支持画笔，图层，图形，艺术字，矢量颜色填充，导出，Google Drive 同步，甚至分享为 JPEG 直链。

### JS Paint - Windows 复古画板风格的在线画板

https://jspaint.app/

> 复刻 Windows 画板的在线画板工具，甚至帮助文档都是  Window CHM 风格的。
